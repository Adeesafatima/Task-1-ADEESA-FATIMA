# To-Do List Application

## Overview

This is a simple command-line To-Do List application developed in Python. The project allows users to manage their daily tasks by adding new tasks and viewing existing ones. It demonstrates the use of fundamental Python concepts such as lists, loops, conditional statements, and user input handling.

## Features

* Add new tasks
* View all saved tasks
* User-friendly menu-driven interface
* Continuous execution until the user chooses to exit

## Technologies Used

* Python 3

## Project Structure

```
todo_list.py
README.md
```

## How to Run

1. Install Python 3 on your system.
2. Download or clone this repository.
3. Open a terminal in the project directory.
4. Run the following command:

```bash
python todo_list.py
```
